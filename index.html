Font Size Units Examples

  • This is 16px (pixels) - Used for fixed-size components that should remain consistent across devices. Example: A button text that needs to maintain exact dimensions.
  • This is 1.5rem (root em) - Used for elements that should scale with user's browser settings. Example: Main heading text that respects user's accessibility preferences.
  • This is 1.2em (relative to parent) - Used for nested elements that should scale relative to their parent. Example: Navigation dropdown menu items that are slightly smaller than the main menu.
  • This is 150% (percentage) - Used for elements that need to be proportionally larger than their container. Example: Highlighted text within a paragraph.
  • This is 3vw (viewport width) - Used for text that should scale with screen size. Example: Hero section headlines that need to look proportional on any device.
  • This is x-large (keyword) - Used for simple relative sizing when exact measurements aren't critical. Example: Warning messages or alerts.
